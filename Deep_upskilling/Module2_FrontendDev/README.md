# Frontend Development Hands-On Exercises
## Cognizant Digital Nurture 5.0

This repository contains the complete implementation of Hands-On exercises for the Frontend Development module as part of the Cognizant Digital Nurture 5.0 Program.

The exercises cover modern frontend development concepts including HTML5, CSS3, JavaScript ES6+, React, Angular, Vue.js, Accessibility, API Integration, and State Management.
