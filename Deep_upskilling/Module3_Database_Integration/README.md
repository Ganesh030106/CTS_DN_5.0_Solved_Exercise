# Database Integration Hands-On Exercises

This folder contains the solved exercises for the Database Integration module in the CTS Digital Nurture program. The material covers PostgreSQL schema design, SQL querying, Python database integration, MongoDB operations, SQLAlchemy ORM, and Alembic migrations.

## Folder Structure

```text

├── Hands_on_1/
├── Hands_on_2/
├── Hands_on_3/
├── Hands_on_4/
├── Hands_on_5/
├── Hands_on_6/
└── Hands_on_7/
```

## Hands-On Summary

### Hands-On 1
Database creation and schema basics.

Contains SQL for creating the college database schema and a normalization write-up that verifies 1NF, 2NF, and 3NF.

### Hands-On 2
Core SQL operations.

Contains SQL exercises focused on creating tables, inserting data, querying records, and validating constraints and relationships.

### Hands-On 3
Advanced SQL practice.

Contains additional SQL exercises that build on joins, filtering, grouping, and analytical queries.

### Hands-On 4
Python database interaction.

Contains a Python-based database exercise that demonstrates working with SQL from a script.

### Hands-On 5
MongoDB / NoSQL exercises.

Contains JavaScript-based MongoDB exercises covering document creation, aggregation, filtering, updates, and delete operations.

### Hands-On 6
SQLAlchemy ORM exercises.

Contains Python ORM examples for model definition, CRUD operations, and solving the N+1 query problem.

### Hands-On 7
Alembic migration management.

Contains SQLAlchemy models, Alembic configuration, generated migration files, and versioning exercises for schema evolution.
