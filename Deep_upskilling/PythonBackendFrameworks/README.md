# Python Backend Frameworks – CTS Digital Nurture 5.0

## Overview

This repository contains the implementation of all hands-on exercises completed as part of the **CTS Digital Nurture 5.0 – Python Backend Frameworks** learning track.

The repository covers:

- Django Framework
- Flask Framework
- FastAPI Framework
- REST API Best Practices
- JWT Authentication & Security
- Microservices Architecture
