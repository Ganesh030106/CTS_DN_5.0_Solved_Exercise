# CTS Digital Nurture 5.0 – Deep Upskilling Track

Welcome to the **Deep Upskilling** repository of the **Cognizant Digital Nurture 5.0 Program**. This repository contains a structured, end-to-end curriculum of hands-on exercises, real-world case studies, design pattern implementations, database integrations, backend web development, and web automation testing.

## 🚀 Environment Requirements & Setup

### Prerequisites
- **Python**: `3.9+`
- **Node.js**: `18.x+` & **npm**
- **Database Systems**: PostgreSQL 14+, MongoDB 6+
- **Browsers & Drivers**: Chrome / Edge with matching WebDriver

### Global Python Virtual Environment Setup
```bash
# Clone the repository
git clone <repository-url>
cd Placement/CTS/Deep_upskilling

# Create and activate Python virtual environment
python -m venv venv
# On Windows PowerShell:
.\venv\Scripts\Activate.ps1
# On Linux/macOS:
source venv/bin/activate
```
