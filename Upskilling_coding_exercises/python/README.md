This folder contains **55 Python coding exercises** covering core Python fundamentals, data structures, object-oriented programming (OOP), file operations, mathematical computations, and algorithmic problem solving.

## 📑 Topics & Concept Mapping

| Exercise Range | Core Python Concepts | Focus Areas |
|---|---|---|
| **Ex 1 – Ex 10** | **Basic Syntax & Variables** | Data Types, String Formatting, Input/Output, Arithmetic Operators, Basic Functions. |
| **Ex 11 – Ex 20** | **Control Flow & Conditionals** | `if-elif-else` branching, Nested Conditionals, Logical Operators (`AND`, `OR`, `NOT`). |
| **Ex 21 – Ex 30** | **Loops & Iteration** | `for` loops, `while` loops, Loop Control (`break`, `continue`), List Iteration, Pattern Generation. |
| **Ex 31 – Ex 40** | **Data Structures** | Lists, Tuples, Dictionaries, Sets, List Comprehensions, Key-Value Searching & Sorting. |
| **Ex 41 – Ex 50** | **Functions & OOP** | Reusable Functions, `*args` / `**kwargs`, Classes, Inheritance, Polymorphism, Encapsulation. |
| **Ex 51 – Ex 55** | **File I/O & Exception Handling** | File Reading/Writing, Try-Except Blocks, Custom Exceptions, String Parsing & Math. |
